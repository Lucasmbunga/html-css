  3. Parágrafos e quebras de linhas
    4. <p>
    Para criar um parágrafo, usamos a tag <p>. E se precisarmos criar vários parágrafos, então temos que usar isso várias vezes.
    <br>
    E para fazer uma quebra de linha, usamos a tag <br> no ponto em que queremos quebrar.
  4. Como inserir imagens e ícone de favoritos
    5. <img>
    Para inserir imagens, usamos a tag <img> que contém dois parámetros, que são: src, que indica a origem da imagem e alt que é um texto alternativo que vai ser exibido no caso em que a imagem não possa ser carregada ou visualizada.

    Também podemos inserir legendas em imagens, usando a tag <figure>, onde dentro desta tag é que iremos inserir a tag da imagem e por sua vez iremos inserir as legendas com a tag <figcaption> como por exemplo:

    <link:favicon>
    E para iserir um ícone de favoritos, usamos a tag <link> dentro do cabeçalho.
  5. Hierarquia de títulos

    6. A hierarquia dos títulos varia de h1 até h6, onde o h1 é o título principal(de maior importância) e o h6(de menor importância)

    <h1>
    A tag <h1> é usado para determinar o título principal.

    E assim os títulos que se seguem vão de h2 até h6 em ordem de importância.

  6. Formatação de Textos

    7. Para formatar um texto são utilizadas várias tags pelas quais:

    <strong>
    A tag <strong> é usada para marcar uma palavra que possui uma força na frase(para negritar).
    <em>
    Usamos a tag <em> para colocar palavras em itálico, ou seja, para dar ênfase nas palavras.
    <ins>
    Usamos a tag <ins> para marcar uma palavra que deve ser lida e prestada a atenção, ou seja, ela sublinha as palavras.
    <del>
    Esta tag é usada para determinar uma palavra deletada ou tachada, ou seja, uma palavra que até possa ser lida, mas deve ser desconsiderada.
    <sup>
    Usamos esta tag para escrever palavras sobrescritas, ou seja, expressões como: X2-5X+6.
    <sub>
    Usamos a tag <sub> para escrever palavras subscritas, ou seja, expressões como: H2O.
  7. Listas em HTML

    8. Existem listas ordenadas, listas não ordenadas e listas de definições.

    1. Listas ordenadas
      2. <ol>(ordered lists)
      Para inserir uma lista ordenada, usamos a tag <ol> para delimitar a lista, e a tag <li>(list item) para identificar cada ítem da lista.

      Esta tag tag também pode conter um atributo type, que pode tomar os seguintes valores: 1,A,a,I,i.

    2. Listas não ordenadas(demarcadas)
      3. <ul>(Unordered lists
      Para criar uma lista não ordenada(com demarcadores), usamos a tag <ul> para delimitar a lista, e a tag <li> para identificar cada ítem da lista.

      Assim como a tag anterior, esta tag também pode conter um atributo type, que pode tomar os seguintes valore: disc, circle e square.

    3. Lista de definições
      4. <dl>(definition list)
      Para criar uma lista de difinições, usamos a tag <dl> para delimitar a lista, e a tag <li> para identificar cada ítem da lista.
